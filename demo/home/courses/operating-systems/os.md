# Operating Systems — Overview

An operating system manages hardware and provides services to programs:
process scheduling, memory management, file systems, and I/O.

- **Process** — a program in execution, with its own address space.
- **Thread** — a lightweight unit of execution sharing the process address space.
- **Scheduler** — decides which ready process runs next (round-robin, priority).
- **Virtual memory** — paging gives each process a large contiguous address space
  backed by physical frames and disk.
